### Introduction

This is a **test implementation guide** for testing GitHub review functionality. It is a minimal FHIR Implementation Guide based on HL7® FHIR® R5.

### Purpose

This Implementation Guide serves as a testing ground for developing a visual review tool that allows users to review content directly on deployed GitHub Pages, with reviews stored in GitHub issues or pull requests.
